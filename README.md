# SparkLink

A beautiful, full-featured URL shortener built with Next.js 16, featuring AI-powered summaries, QR code generation, click analytics, and a sleek dark UI.

## Features
- Instant URL shortening
- AI-generated page summaries (mock for demo)
- Dynamic QR codes
- Real-time analytics simulation
- Local persistence with localStorage
- Framer Motion animations
- Responsive design

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Lucide Icons
- Framer Motion
- QRCode.js

## How to Run
```bash
npm install
npm run dev
```
